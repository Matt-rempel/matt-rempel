The 2020 Project

  • REST API
  • Asynchronous loading of images and content
  • User data sync between devices through sign in partners or iCloud

The official iOS App of The 2020 Project. I was contracted by Foothills Alliance Church to build this iOS App. It will load a list of devotions which is updated daily. You will be able to track your progress of how many you read and have it sync between devices with an account. There are also features such as search, daily reminders, and bookmarks to help bring a richer experience. All of the source code for this App is on my GitHub which is liked below.

Technical Features

This App calls a REST API to populate the App's data. This REST API has endpoints for the most recent devotions, individual devotions, searching the devotions and much more. This App also uses endpoints with OAuth2 for account management. This allows use progress, bookmarks and other data to be synced between the iOS, Android and Web apps.

Technical Details

  • UI Framework: UIKit
  • Language: Swift

Assurance Construction

  • Automatic location based time log entry
  • Firebase
  • User accounts for each employee
  • Download safety documentation to the device

The official internal iOS App of Assurance Construction. I was contracted by Assurance Construction to build this iOS App. It will allow employees of the company to view Jobs, Time Logs, Snow Logs, Snow Sites, Safety Documentation, Track equipment and use automatic time log entry based on device location. Employees can view jobs assigned to them, add time log hours, track their equipment usage and ensure they have all the required safety documentation for operating machinery on hand.

Technical Features

This App connects to Firebase as it's backend service. It will use background location tracking to detect when the user enters a specified location that is assigned to them through the app on an administrators device. The app will then send a push notification to the user allowing them to open the app and confirm the time log entry, along with adding photos or notes to that entry.

Technical Details

  • UI Framework: UIKit
  • Language: Swift

This project is not public for security reasons

Maintenance iOS App

  • Guides employee through the maintenance process
  • Auto generated PDF of the maintenance for clients

Employees of Benjamin's Aquarium Services carry iPads to their client maintenance appointments. Living on the iPad is an App I developed which walks the employee through the entire maintenance process. They can enter data and photos collected through out the maintenance which when finished will generate a PDF that is sent to the client.

Technical Features

This App connects to Firebase as it's backend service. All maintenances, clients, photos, and merchandise are saved to Firebase to cross device syncing. PDF generation is done with a library that converts UIViews to PDF pages.

Technical Details

  • UI Framework: UIKit
  • Language: Swift

This project is not public for security reasons
